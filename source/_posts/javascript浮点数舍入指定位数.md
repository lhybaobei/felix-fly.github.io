下面是一个小例子，方法很简单，可以将浮点数四舍五入为指定位数的浮点数


```<script type="text/javascript">
var a = 3.145;
alert(a.toFixed(2));
var b = 3.141;
alert(b.toFixed(2));
</script>
```
